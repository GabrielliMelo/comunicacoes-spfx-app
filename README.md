# SX TEMPLATE SPFX

Projeto template para reuso.

## Tecnologias Utilizadas

- **[Tailwind CSS](https://tailwindcss.com/)**: Uma framework CSS de utilidade-primeira para estilos rápidos e responsivos.
- **[Headless UI](https://headlessui.dev/)**: Um conjunto de componentes de interface de usuário completamente acessíveis e sem estilo, projetados para integrar-se perfeitamente com Tailwind CSS.
- **[Heroicons](https://heroicons.com/)**: Coleção de ícones SVG finamente ajustados para interfaces de usuário modernas, desenhados pela equipe do Tailwind Labs.
- **[Moment.js](https://momentjs.com/)**: Biblioteca para analisar, validar, manipular e formatar datas.
- **[TypeScript](https://www.typescriptlang.org/)**: Superset de JavaScript que adiciona tipagem estática e recursos de programação orientada a objetos.

## Requisitos

- **Node.js**: Este projeto requer Node.js versão **18**. Certifique-se de que esta versão está instalada antes de prosseguir com a configuração do projeto.
